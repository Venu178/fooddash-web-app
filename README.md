# fooddash-web-app
A modern food delivery web application featuring a responsive landing page, promotional grids, dynamic restaurant filtering, and a seamless checkout workflow.
## Features & Architecture

### 1. Hero Section & Value Proposition
* **The Hook:** A bold, high-contrast headline ("Delicious food, delivered fast") paired with a subtext explaining the core service.
* **Call to Actions (CTAs):** Primary orange button for *Browse Restaurants* and a secondary black button for *Search Food*.
* **Social Proof:** Displays 8+ Restaurants, 48+ Dishes, and a 4.6★ Avg Rating.

### 2. Promotional & Offers Section
A vibrant, multi-colored grid layout featuring active promotional codes:
* **FIRST50:** 50% OFF for first-time users.
* **FREEDEL:** Free Delivery on orders over $25.
* **BOGO:** Buy 1 Get 1 deal on all desserts.

### 3. Food Categories & Exploration
* **Cuisine Filters:** Quick-tap filter buttons for popular food types (Pizza, Burgers, Sushi, Chinese, Indian, Desserts, Drinks, and Salads).

### 4. Featured Restaurants Grid
* **Badges & Metrics:** Utilizes a BESTSELLER tag, star ratings, and review counts (e.g., Bella Napoli: 4.8 rating from 324 reviews).
* **Metadata:** Displays cuisine tags, delivery duration (25-35 min), delivery fees ($2.99), and minimum order requirements ($15 min).

### 5. "How It Works" & Footer Architecture
* **Onboarding Steps:** 3-step breakdown (Browse & Search → Add to Cart → Get Delivered).
* **Footer:** 3-column corporate links structure (Company, Support, and Legal) along with social media anchors.
